BDC（Battle Damege Calc）公開手順（Cloudflare Pages / 手動アップロード）
1) Cloudflare ダッシュボード → Workers & Pages → Create application → Pages
2) 「Drag and drop」オプションを選択し、このフォルダ（ZIPを解凍した中身）を丸ごとドラッグ&ドロップ
3) プロジェクト名（例：bdc）を入力 → Deploy
4) デプロイ後は pages.dev のURLが発行されます（カスタムドメインも後で設定可）
5) 更新時：Pagesの「Deployments」→「Upload new assets」から同じ手順で上書き

※ 免責（DISCLAIMER.txt）とプライバシー（PRIVACY.txt）のリンクをサイト内（フッター等）から案内してください。
※ 広告や解析を導入する場合は各事業者の指示に従ってください（本パックには実装を含みません）。
